Baraza

Meetings and Video Conferencing

A video conferencing and collaboration software that enables users to host audio and video meetings, share files, direct message each other, and participate in group chats. It also had a polling feature where users could vote on polls within a specified group.

I worked on the REST API endpoints with Python and Django for meetings, teams, user polling, and the back-end integration with OpenVidu. Additionally, I implemented automated, asynchronous, and periodic tasks using celery for recurring meetings scheduling and email and mobile notifications.
